# Portfolio | Nicoo - Nicolas Osborne - Développeur Front-End & Intégrateur Web - Multi-OS

Un projet de portfolio personnel présentant mon parcours, mes réalisations et mes compétences en Développement et Intégration Web.

# Stack technique :

- Next.js
- TypeScript
- SCSS

# Lancer le projet :

## 1️⃣ Cloner le dépôt

```bash
- git clone https://github.com/NicolasOsborne/nicoo-portfolio.git
- cd nicoo-portfolio

```

## 2️⃣ Installer les dépendances :

```bash
- npm install
ou
- yarn
```

## 3️⃣ Lancer le projet :

```bash
- npm run dev
ou
- yarn dev
```

## 🌐 Accéder à l'application

L'application sera accessible à l'URL : http://localhost:3000

# Accéder à la version en ligne du portfolio

Le projet a été déployé à l'aide de Vercel, et est consultable à l'URL suivante : https://nicoo-portfolio.vercel.app/fr
